# CyberlifeCN.github.io
Host on GitHub

### 怎样将域名绑定到 github pages 博客上
1. 创建仓库 CyberlifeCN.github.io
注意仓库名 github.io 的前缀必须是 GitHub 上的用户名
2. 添加CNAME文件
在仓库根目录下添加一个CNAM文件，没有后缀名，里面内容为你的域名(如:cyber-life.cn),不需要添加http/www等前缀。
